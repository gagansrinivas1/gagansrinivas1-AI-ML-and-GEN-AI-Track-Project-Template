# Brainstorming & Ideation

This folder holds brainstorming notes, idea sketches, early proposals, and other ideation artifacts for the project.

Suggested files to add here:
- ideas.md — raw idea list and short descriptions
- notes/<date>-meeting-notes.md — meeting notes and action items
- sketches/ — diagrams or concept images

Please add your notes and organize subfolders as needed.
